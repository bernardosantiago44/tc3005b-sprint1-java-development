# Order Management System - Sprint 1 Boilerplate

Practice boilerplate for **Exceptions, Logging & Unit Testing** in Java.

## Quick Start

```bash
# Compile
mvn compile

# Run tests
mvn test

# Coverage report
mvn test jacoco:report
open target/site/jacoco/index.html
```

## Instructions

Read the [INSTRUCTIONS.md](INSTRUCTIONS.md) file for the full deliverable instructions.

## Structure

- `src/main/java/` — Source code with TODOs to implement
- `src/test/java/` — Tests with TODOs to complete
- `src/main/resources/logback.xml` — Logging configuration
